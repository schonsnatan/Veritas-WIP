# Veritas: Structured Mental Wellness Orchestrator

> **Status:** 🚧 **Work In Progress (WIP)** 🚧  
> This project is currently under active development. 

## What is Veritas?

**Veritas** is not just another AI chatbot. It is an orchestration system that leverages **RAG (Retrieval-Augmented Generation)** to provide mental wellness recommendations grounded in structured data.

The project’s differentiator is the use of **validated screening instruments** (such as PHQ-9 and GAD-7) to structure user input. This ensures that information retrieval, whether from clinical papers or philosophical texts, is precise, safe, and contextually relevant.

## Privacy-First
By utilizing **Ollama**, all sensitive psychological data is processed locally. This project adheres to a "Privacy-by-Design" philosophy, ensuring that personal reflections never leave the user's infrastructure.

## Important Disclaimer

**Veritas** is a portfolio project for demonstrating AI Engineering and **is not a substitute for professional medical or therapeutic advice**. The system focuses on wellness and informative support based on public bibliographies.
